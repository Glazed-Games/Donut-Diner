# Donut Diner

First-person adventure prototype developed in Unity as a team project.

## Overview

Donut Diner is a prototype focused on building the core systems for a first-person interactive experience.

## Systems Implemented

- First-person movement and camera control
- Player state machine (idle, walk, crouch, fall, carry, inspect)
- Raycast-based examination and interaction system
- Interactive objects such as doors, NPCs, and devices
- Item system with carryable, inspectable, and pickup items
- Item placement spots connected to environmental interactions
- Item inspection system with rotation, zoom, and dedicated UI
- Portal system with camera rendering and teleportation
- Basic game state and UI state management
- Serialization support for persistent object states

## Tech Stack

- Unity
- C#

## Purpose

The project was developed as a collaborative prototype for a first-person adventure game, with a focus on system design, interaction, and extensibility.
